⚙️ SteamCMD — установка сервера

Создайте папку C:\L4D_Server. Скачайте steamcmd.zip с valve. Внутри выполните:

login anonymous
force_install_dir C:\L4D_Server
app_update 222840 validate
quit

update.bat для быстрого обновления:

steamcmd +login anonymous +force_install_dir . +app_update 222840 validate +quit
✅ AppID 222840 = L4D1 dedicated server. Размер ~8 ГБ, время загрузки ~5-20 мин.

🖥️ Первый запуск (srcds.exe)

Создайте start_server.bat в C:\L4D_Server:

@echo off
cd /d C:\L4D_Server
srcds.exe -console -game left4dead -port 27015 +map l4d_hospital01_apartment +sv_lan 0 -secure
-tickrate 100 высокий тикрейт (с плагином)
-maxplayers 12 больше слотов
-nomaster скрыть из списка
-ip 192.168.x.x привязка IP

🛠️ server.cfg — ядро сервера

Путь: left4dead\cfg\server.cfg. Минимальная рабочая конфигурация:

hostname "Мой L4D1 Server"
rcon_password "securepass2025"
sv_password ""
sv_lan 0
sv_region 4
sv_maxrate 25000
sv_minrate 10000
sv_maxupdaterate 30
z_difficulty Normal
maxplayers 8
mp_afk_time 45
sv_allow_lobby_connect_only 0
exec banned_user.cfg

RCON: удаленное управление через rcon_password. Не забудьте сменить пароль!

🎨 autoexec.cfg + контуры (GLOW)

Файл для клиента Steam/left4dead/left4dead/cfg/autoexec.cfg. Улучшает интерполяцию, прицел и цвета:

rate "30000"
cl_cmdrate "30" | cl_updaterate "30"
cl_interp "0.067" | cl_interp_ratio "2"
// ===== ЦВЕТА КОНТУРОВ (лучшая видимость) =====
cl_glow_infected_r "0.0"
cl_glow_infected_g "0.5"
cl_glow_infected_b "1.0" // синие зараженные
cl_glow_item_r "0.60"
cl_glow_item_g "0.46"
cl_glow_item_b "1.00" // фиолетовые предметы
cl_glow_survivor_vomit_r "0.9"
cl_glow_survivor_vomit_g "0.0"
cl_glow_survivor_vomit_b "0.8"
bind "F9" "thirdpersonshoulder"
🎯 Совет: cl_crosshair_dynamic 0 и настройка толщины прицела.

🧩 Metamod:Source + SourceMod

  1. Скачайте Metamod:Source (Windows) и SourceMod (Windows).
  2. Распакуйте Metamod в left4dead → папка addons.
  3. Сгенерируйте metamod.vdf для Left 4 Dead на metamodsource.net/vdf.php и замените.
  4. Распакуйте SourceMod в left4dead (перезапись).
  5. Права админа: узнать SteamID через status, добавить в addons/sourcemod/configs/admins_simple.ini:
    "STEAM_1:1:12345678" "99:z"
📌 Проверка: в консоли сервера напишите meta list — должен показать SourceMod.

🌐 Проброс портов и приватный режим

Брандмауэр: UDP 27015,27016,27017
Роутер: Forward UDP 27015 → IP ПК
Приватный сервер (по IP): в start_server.bat добавьте -nomaster, в server.cfg → sv_tags "hidden" и sv_allow_lobby_connect_only 1.

⚡ Продвинутые моды (32 слота, 100 тикрейт)

Скачайте L4DToolZ (l4d1 версию), распакуйте в addons. В start_server.bat укажите -maxplayers 14, в server.cfg: sm_cvar sv_maxplayers 14 и sm_cvar survivor_limit 14.

Установите плагин Tickrate_Enabler, в параметры запуска добавьте -tickrate 100, в server.cfg: sv_maxupdaterate 100; sv_maxcmdrate 100.

Карты с L4DMaps кладём в left4dead/addons. В server.cfg: sv_allowdownload 1; sv_allowupload 1; net_maxfilesize 100. Смена карты: changelevel название_карты.

📊 Мониторинг, HLSW, бэкапы и авторестарт

HLSW — утилита для мониторинга сервера (RCON, статус).

Авторестарт при краше: используйте скрипт watchdog или FireDaemon.

:loop
srcds.exe -console -game left4dead -port 27015 +map l4d_hospital01_apartment
timeout /t 10
goto loop

Бэкап конфигов и плагинов
Создайте архив left4dead/cfg и addons/sourcemod/configs раз в неделю.

SourceBans++ — веб-панель для управления банами и админами.

Производительность: используйте sm_cvar fps_max 0 и приоритет процесса "Высокий".

❓ Частые проблемы + новые решения

Порт занят. Поменяйте в bat: -port 27016 и откройте UDP в брандмауэре.

Обновите Metamod.vdf через генератор сайта metamodsource.net. Проверьте addons/metamod/bin.

Проверьте проброс порта, sv_lan 0, брандмауэр для srcds.exe. Для теста используйте heartbeat в консоли.

Убедитесь что параметр -secure в bat и secure 1 в server.cfg. Временное решение: -insecure для локальных тестов.

Удалите старые версии карт .vpk из addons у клиента и сервера, синхронизируйте файлы.

Включите Tickrate Enabler, увеличьте sv_maxupdaterate 66, установите L4DToolZ с оптимизациями, выключите лишние плагины.

🚀 Запуск и подключение

  1. Запустите start_server.bat. Дождитесь строки "Connection to Steam servers successful".
  2. В клиенте L4D1 откройте консоль ` и введите connect 127.0.0.1 (локально) или ваш внешний IP.
] connect 192.168.1.10:27015
Connecting to 192.168.1.10:27015...
Готово! Для друзей используйте whatsmyip и раздайте IP:порт.
Совет PRO: для защиты от флуда RCON добавьте в server.cfg → sv_rcon_banpenalty 10 и sv_rcon_maxfailures 5. Также можно использовать L4D1 Community Fixes плагин для исправления вылетов.
